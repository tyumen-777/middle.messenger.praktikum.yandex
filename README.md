# Веб мессенджер на Vanilla JS + Handlebars

---
## Технологии

- #### Typescript

- #### Handlebars

## Установка

- `npm install` — установка стабильной версии,
- `npm run dev` — запуск версии для разработчика,
- `npm start` — сборка стабильной версии и запуск на 3000 порту.
